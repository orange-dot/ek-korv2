# Brate,

Napravio sam demo za HAX aplikaciju. Moraš da vidiš.

## Setup

```bash
git clone https://github.com/user/autobusi-punjaci
cd autobusi-punjaci/web
npm install
npm run dev
```

http://localhost:5173/hax

## Šta gledaš

- **84 physics modela** @ 10,000 ticks/sec
- **50ms fault recovery** - ubaci fault, gledaj kako se mesh reformira
- **V2G** - moduli reaguju na grid frequency u realnom vremenu

Desni panel - interactive controls. Igraj se sa power, frequency, temperature, fault injection.

"Simulation before silicon" - SpaceX pristup. Pre nego što potrošimo pare na hardver, sve testiramo u simulaciji.

Javi kad pogledaš.
