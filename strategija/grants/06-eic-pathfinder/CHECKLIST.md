# EIC Pathfinder - Application Checklist

**Program:** EIC Pathfinder (Open or Challenge)
**Amount:** €3-4M (consortium)
**Status:** BACKUP OPTION
**Use When:** If EIC Accelerator unsuccessful

---

## When to Apply

EIC Pathfinder is appropriate if:
- [ ] EIC Accelerator not successful
- [ ] Need fundamental research funding
- [ ] Want academic consortium
- [ ] Technology needs more validation (TRL 1-4)

## Prerequisites

- [ ] Identify academic partners
- [ ] Confirm research questions are novel
- [ ] Check current Pathfinder calls (Open vs Challenge)
- [ ] Assess consortium readiness

## Application Components

### Short Proposal (Step 1)

- [ ] Project abstract
- [ ] S&T excellence summary
- [ ] Impact summary
- [ ] Consortium description
- [ ] Budget overview

### Full Proposal (If Invited)

- [ ] Detailed S&T methodology
- [ ] Work packages and Gantt
- [ ] Detailed budget per partner
- [ ] CVs of key personnel
- [ ] Ethics self-assessment

## Key Differences from Accelerator

| Aspect | Pathfinder | Accelerator |
|--------|------------|-------------|
| Focus | Research | Commercialization |
| TRL | 1-4 | 5-8 |
| Applicant | Consortium | Single SME |
| Amount | €3-4M | €2.5M grant |
| Duration | 3-4 years | 2 years |
