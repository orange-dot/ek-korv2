# WBSO Netherlands - Application Checklist

**Agency:** RVO

## Prerequisites

- [ ] Netherlands BV registered (KvK number)
- [ ] eHerkenning account (level 2+)
- [ ] Bank account in company name
- [ ] At least one employee on payroll (including directors)

## Application Preparation

### Administrative

- [ ] KvK registration number ready
- [ ] Company fiscal year confirmed
- [ ] Payroll administration set up
- [ ] eHerkenning credentials obtained

### Technical

- [ ] R&D project description written (APPLICATION.md)
- [ ] Hour estimate per activity
- [ ] Personnel assignment
- [ ] Technical uncertainty documented

## Application Process

### Step 1: Portal Access
- [ ] Go to mijn.rvo.nl
- [ ] Log in with eHerkenning
- [ ] Navigate to WBSO section

### Step 2: Application Form
- [ ] Enter company details
- [ ] Select application period
- [ ] Enter project title and description
- [ ] Detail R&D activities
- [ ] Specify personnel and hours
- [ ] Calculate expected benefit

### Step 3: Submit
- [ ] Review all entries
- [ ] Submit application
- [ ] Save confirmation number

## Post-Submission

- [ ] Set up hour tracking system
- [ ] Inform team about tracking requirements
- [ ] Calendar reminder for realization report

## Ongoing Obligations

### Monthly
- [ ] Track R&D hours per project
- [ ] Document activities
- [ ] Save evidence (code commits, docs, etc.)

### End of Period
- [ ] Compile realization report
- [ ] Calculate actual hours
- [ ] Submit to RVO
- [ ] Claim benefit in wage tax return
