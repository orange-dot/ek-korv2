# ELEKTROKOMBINACIJA

Modularna EV infrastruktura za punjenje flota električnih autobusa.

## Aktuelni Koncept: EK3 (3kW Moduli)

Univerzalni 3kW power moduli koji se kombinuju za bilo koju primenu - od 3kW do 3MW.
